FEATURES

Screen Capture

Record a tab, window, or your whole display via the browser's native capture API.

Mic Audio

Capture your voice through the microphone, with graceful fallback to system audio.

Pause / Resume

Stop the action mid-take and pick it back up — the timeline stays continuous.

WebM Export

Preview and download the footage locally as a standard .webm file. No uploads.

HOW IT WORKS
1

Start

Hit Start, pick what to share, and grant mic access.

2

Record

Use Pause / Resume anytime; press Stop when done.

3

Export

Preview the reel, then Download or Delete the footage.

DOCUMENTATION

Screen capture requires a secure context, so serve the app over localhost or https rather than opening the file directly.

urGlimpse — run 
# From the project folder
python -m http.server 8000
# then open http://localhost:8000
  • Records use the MediaRecorder, getDisplayMedia & getUserMedia APIs.
  • Footage is processed entirely in your browser — nothing is uploaded.
  • Works best in Chromium-based browsers (Chrome, Edge, Brave).
